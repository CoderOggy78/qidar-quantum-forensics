Pipeline

Five stages, one forensic chain of custody

Each module hands a cryptographically verifiable artifact to the next — detection becomes evidence, evidence becomes lineage.

01

Quantum-Assisted Anomaly Detection Q-AAD

Maps appliance telemetry into a quantum Hilbert space as a superposed state vector, then fuses a classical neural readout with a unitary-rotated quantum measurement to score anomalies in real time.

A(I) = σ(w·I + b) · Re⟨E|U†MU|E⟩
02

Adaptive Blockchain Ledger Forensics ABLF

Every flagged event is hashed into a forward-linked block and validated by 2-of-3 smart-contract consensus — a decentralized, zero-trust audit trail that resists silent tampering.

B(t) = SHA3( FD(t) || T(t) || B(t−1) )
03

Multi-Modal AI Forensic Reconstruction MMAFR

Sensor telemetry, control commands, and network logs are fused through attention-guided memory networks to chronologically rebuild the attack sequence and surface hidden attack vectors.

L = Σ‖f_r(l¹,…,lʲ) − X‖²
04

Quantum-Enhanced Cryptographic Logging QECL

Von Neumann entropy from a measured quantum state seeds a probabilistic key stream, sealing forensic logs behind a quantum one-time pad resistant to classical and quantum adversaries alike.

S = −tr(d·log d) → key → E(I_F) = key ⊕ I_F
05

Blockchain-Integrated Quantum Provenance Chain BIQPC

Anomaly, timestamp, source, and signature are anchored into a quantum-hash provenance ledger, verified by zero-knowledge proofs — full lineage, zero data disclosure.

Q_p = f_h[ Q_event ⊕ f_h(e_t, S_t, δ) ]

demo.py 
from src.pipeline import QIDARPipeline
import numpy as np

pipeline = QIDARPipeline(anomaly_threshold=0.06)

result = pipeline.process_event(
    device_id="kitchen-hub-03",
    feature_vector=np.array([0.9, 0.8, 0.7, 0.6, 0.5, 0.4]),
)

print(result["is_anomaly"])       # True
print(result["block_hash"])      # SHA-3 forensic block
print(result["provenance_anchor"]) # BIQPC quantum anchor

audit = pipeline.audit()
print(audit)  # {'ledger_valid': True, 'provenance_valid': True}
